Review4You! is a python web application powered by the Flask microframework. This application acts as a one stop shop for looking up businesses near you, paired with an intuitive GoogleMap interface. Searches are powered by the Yelp Fusion API. Review4You! is hosted on the Heroku cloud hosting platform.

The instructions are simple, just input your zip code along with some search parameters and watch the GoogleMap be populated with markers of which are constructed based on your results!

You can see Review4You!'s source code and other projects on my GitHub
